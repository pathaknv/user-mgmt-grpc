# user-mgmt-grpc
Golang based grpc client &amp; server for user management

# Running the app locally:
- Clone the repo locally
- Terminal 1:

  `cd <repo-directory>/client`
  
  `go run .`
- Terminal 2:

  `cd <repo-directory>/server`
  
  `go run .`
- Terminal 3:

  `cd <repo-directory>`
  
  `docker-compose up db`
